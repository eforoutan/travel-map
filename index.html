Places I've Been

An interactive world map for pinning everywhere you've wandered — search a city or right-click to drop a marker, label it, theme it, and watch your travels spread across the globe.

Leaflet.js
The leading open-source web-mapping engine — handles the interactive map, markers, and smooth pan/zoom. No API key required.
CARTO Voyager tiles
Clean, modern raster basemap tiles served over OpenStreetMap data.
OpenStreetMap · Nominatim
Free worldwide geocoding & reverse-geocoding — turns city searches and map clicks into coordinates and place names.
Vanilla JS · HTML · CSS
No frameworks, no build step. CSS custom properties power the six switchable themes.
Shared places.json + localStorage
Pins load from a shared places.json in the repo so every device sees the same map. Local edits are cached in your browser until you publish.
Manrope
A clean, modern sans-serif from Google Fonts for the interface typography.

A single self-contained index.html — drop it on GitHub Pages and it just works.

Publish your map

One file downloaded

A file called places.json just downloaded. To make these pins show up for everyone on every device, commit that file to your repo:

  1. Open your repo on GitHub (the same one with index.html).
  2. Click Add file → Upload files.
  3. Drag in the downloaded places.json. If it asks to replace the existing one, allow it.
  4. Click Commit changes.

GitHub Pages redeploys in a minute or two. After that, reloading the live site on any device shows the same pins. Until you publish, your pins live only in this browser.
